# Security

Drift is a local desktop audio application. It does not ship with network services or accounts.

If you discover a security issue (for example, unsafe deserialization of patch files, or a crash that could indicate memory corruption), please report it responsibly:

1. Open a **private** security advisory on GitHub for this repository (if enabled), or  
2. Email or message the maintainers with enough detail to reproduce the issue.

Please avoid posting exploit details in public issues until a fix is available.
